# Prompt-Driven Playable Ad Framework
This graduation project aims to develop a comprehensive playable ad framework. Playable ads require specific, reusable components. The repository is structured to provide these core modules out-of-the-box. A key innovative feature of the system is its LLM-ready architecture, allowing for the automated generation of entire game prototypes from simple natural language prompts.
